# Machine-Learning-Web-Apps
Building and Embedding Machine Learning Model into a Web App(With Flask,Streamlit,Express,etc)


### Basic Requirements
#### For Python ML Web Apps
+ Flask
+ Streamlit
+ Scikit-learn
+ Pandas
+ Numpy
+ Joblib/Pickle
+ Matplotlib
+ Spacy
+ NLTK
+ Textblob
+ Requests


#### For Nodejs ML Web App
+ Express.js
+ Brainjs
+ Body-parser
